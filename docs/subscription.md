## Employee Management?
  Employee Management allow employees to enter their attendence or work log based on that salary is calculated and employees have their own login to enter their work log or attendance.
## Employee User Control?
   Employee User Control give power to the existing employee to manage the modules that are avaliable for admin eg: Manager, Supervisor, sales person ect... this allow the specific employee to access specific customer/vendor and specific actions into it.
## Custom ACL for Users (roles)
   This allow the Admin to create custome access controle for the "Employee User Control" without "ACL" admin can not "Employee User Control" for portal. eg: allow to update order status to invoice, allow to record payment receive ect... totally custom
